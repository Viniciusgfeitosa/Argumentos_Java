# 📌 Argumentos em Java

Projeto simples em Java para demonstrar o uso de argumentos de linha de comando no método main.

# 🚀 Objetivo

Aprender como passar valores ao executar um programa Java via terminal e utilizá-los dentro do código através do parâmetro:

String[] args

# 💻 Código do projeto
public class Argumentos {

    public static void main(String[] args) {
    
        System.out.println("Você digitou " + args[0]);
    }
    
}

# 💻 Exemplo de funcionamento

Ao executar o programa com:

java Argumentos teste

Saída:

Você digitou teste

# 🧠 Conceitos abordados

Método main

Parâmetro String[] args

Argumentos de linha de comando

Compilação com javac

Execução com java

# 📚 Finalidade

Este projeto foi desenvolvido para fins de estudo, com foco em iniciantes que estão aprendendo Java e execução via terminal.
